# API Tester Backend

This is the **backend** of the API Tester application, built with **Node.js** and **Express**, deployed on **Render**. It provides API endpoints for testing and responding to HTTP requests from the frontend.

## Live API

- **Backend API:** [https://api-tester-backend-3qmj.onrender.com](https://api-tester-backend-3qmj.onrender.com)  
- **Frontend Application:** [https://api-tester-app-ochre.vercel.app](https://api-tester-app-ochre.vercel.app)

## Features

- Handle **GET**, **POST**, **PUT**, and **DELETE** requests.
- Sends back response **status codes**, **headers**, and **JSON data**.
- Serves as the backend for the **API Tester frontend**.

## Tech Stack

- **Backend:** Node.js, Express  
- **Database:** Supabase 
- **Hosting:** Render
